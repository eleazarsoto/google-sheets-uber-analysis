# 🚗 Uber Rides Operational Analysis
### Google Sheets Project | Eleazar Soto

## 📋 Project Overview
Operational analysis of Uber ride data including booking status, 
vehicle types, payment methods, ratings and cancellations.
Large-scale dataset with 150,000 records analyzed using 
Pivot Tables and dynamic reports.

## 📊 Dataset
- **Records:** 150,000 Uber rides
- **Variables:** 23 columns
- **Includes:** Booking status, vehicle type, locations, ratings, 
payment methods, cancellations

## 🔍 Key Questions Analyzed
- What percentage of rides are completed vs cancelled?
- Which vehicle types are most popular?
- What payment methods do customers prefer?
- What are the average driver and customer ratings?
- Which locations have the highest demand?

## 🛠️ Tools & Techniques
- Google Sheets
- Pivot Tables
- Charts and Visualizations
- Data Cleaning

## 💡 Key Findings
- **62.3%** of rides completed successfully
- **17.9%** cancelled by driver — highest cancellation source
- Most popular vehicle: **Auto** (25%)
- Most used payment method: **UPI** (44.7%)
- Average booking value: **$511.24**
- Average ride distance: **24.6 km**
- Top pickup location: **AIIMS**
- Driver ratings and customer ratings are closely matched (~43/100)
